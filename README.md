PCFG-parser

https://en.wikipedia.org/wiki/Cyk_algorithm 
https://en.wikipedia.org/wiki/Inside%E2%80%93outside_algorithm
https://courses.cs.washington.edu/courses/cse590a/09wi/pcfg.pdf
